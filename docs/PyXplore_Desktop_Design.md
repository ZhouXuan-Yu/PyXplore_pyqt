# PyXplore Desktop 应用架构设计方案

## 1. 项目概述

**项目名称**: PyXplore Desktop  
**项目类型**: 桌面端X射线衍射分析软件  
**核心功能**: 整合PyXplore库的所有功能，提供图形化界面，支持单文件处理和批量处理  
**目标用户**: 材料科学研究者、XRD实验人员、结构分析工程师

---

## 2. 系统架构设计

### 2.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | PyQt5 (5.15.x) |
| 图表展示 | Matplotlib + FigureCanvasQTAgg |
| 数据处理 | NumPy, Pandas |
| 底层计算 | PyXplore (src模块) |
| 打包工具 | PyInstaller / cx_Freeze |

### 2.2 模块架构

```
PyXploreDesktop/
├── main.py                 # 应用入口
├── app/
│   ├── __init__.py

│   ├── config.py           # 配置管理

│   ├── background/         # 背景扣除

│   ├── xrd_simulation/    # XRD模拟
│   ├── xrd_refinement/    # XRD精修
│   ├── amorphous/          # 非晶分析
│   ├── xps/               # XPS分析

│   └── batch/              # 批量处理

    ├── icons/              # 图标

```

### 3.1 主窗口设计

**结构**: QMainWindow + QSplitter + QTabWidget

```
┌─────────────────────────────────────────────────────────────────┐
│  PyXplore Desktop                                    [─][□][×]  │

│            │  ┌──────────────────────────────────────────────┐  │

│  📁 CIF预处理│  │            内容区域                           │  │
│  📁 XRD模拟  │  │    (根据左侧选择显示不同模块)                  │  │
│  📁 XRD精修  │  │                                              │  │
│  📁 非晶分析 │  │                                              │  │
│  📁 XPS分析  │  └──────────────────────────────────────────────┘  │
```

**左侧导航**: QTreeWidget 实现，可折叠的模块树  
**内容区域**: QStackedWidget + 多个功能页面  
**底部状态栏**: QStatusBar，显示运行状态和日志

### 3.2 各功能模块页面设计

#### 3.2.1 背景扣除模块

**功能**: 对XRD原始数据进行背景扣除

**界面元素**:
- 文件输入区: 
  - 原始数据文件选择 (CSV/TXT/XRDML)
  - 数据预览表格
- 参数设置区:
  - 低频滤波比例 (Slider: 0.1-1.0)
  - 背景点数 (SpinBox)
  - 分段数 (SpinBox)
  - SG滤波窗口 (SpinBox: 奇数)
  - 多项式阶数 (SpinBox)
  - 谱图类型选择 (ComboBox: XRD/Raman/XPS)
- 预览区:
  - 原始数据 vs 背景曲线对比图
  - 去背景后的数据预览
- 操作按钮: "开始处理", "保存结果", "重置"

**调用函数**:
```python
WPEM.BackgroundFit(
    intensity_csv,
    LFctg=0.5,
    bac_num=None,
    bac_split=5,
    window_length=17,
    polyorder=3,
    Model='XRD'
)
```

#### 3.2.2 CIF预处理模块

**功能**: 读取CIF文件，计算晶体学参数

**界面元素**:
- 文件输入: CIF文件选择
- 参数设置:
  - 波长选择 (ComboBox: CuKa, CoKa, MoKa, 自定义)
  - 2θ范围 (Min/Max输入框)
  - 显示晶胞选项 (CheckBox)
  - 计算消光选项 (CheckBox)
  - 结构弛豫选项 (CheckBox) - 需要M3Gnet
- 输出显示:
  - 晶格常数表格 (a, b, c, α, β, γ)
  - 原子坐标表格
  - 晶胞密度
- 操作按钮: "解析CIF", "导出参数"

**调用函数**:
```python
WPEM.CIFpreprocess(
    filepath,
    wavelength='CuKa',
    two_theta_range=(10, 90),
    show_unitcell=False,
    cal_extinction=True,
    relaxation=False
)
```

#### 3.2.3 XRD模拟模块

**功能**: 从CIF文件模拟XRD图谱

**界面元素**:
- 结构输入:
  - CIF文件选择
  - 或手动输入晶格常数和原子坐标
- 参数设置:
  - 波长选择
  - 2θ范围和步长
  - 超胞设置 (CheckBox + 三个方向倍数)
  - 固溶体替换 (元素1, 元素2, 比例)
  - 空位设置 (原子类型, 空位比例)
  - 晶粒尺寸 (nm)
  - 择优取向参数
  - 热振动参数
  - 零点偏移
  - 背景模拟选项
- 预览区:
  - 模拟XRD图谱
  - HKL指标化结果
- 操作按钮: "开始模拟", "保存数据", "导出图像"

**调用函数**:
```python
WPEM.XRDSimulation(
    filepath,
    wavelength='CuKa',
    two_theta_range=(10, 90, 0.01),
    SuperCell=False,
    PeriodicArr=[3,3,3],
    ReSolidSolution=None,
    RSSratio=0.1,
    GrainSize=None,
    orientation=None,
    thermo_vib=None,
    zero_shift=None,
    bacI=False
)
```

#### 3.2.4 XRD精修模块

**功能**: WPEM全谱拟合和晶格常数精修

**界面元素**:
- 数据输入:
  - 去背景数据文件
  - 原始数据文件
  - 背景数据文件
  - CIF文件(可多个)
- 精修参数:
  - 波长设置 (单波长/双波长)
  - 2θ范围
  - PV函数参数 (bta, bta_threshold)
  - 收敛参数 (limit, iter_limit, w_limit)
  - 最大迭代次数
  - 晶格常数约束 (固定/自由)
  - 质量分数计算方式 (近似/精确)
  - CPU核心数
- 峰选择参数:
  - 子集峰数
  - 角度范围
- 高级选项:
  - 加载已有参数
  - 零点校正
  - 大分子模式
- 输出显示:
  - 精修曲线对比图
  - R因子 (Rp, Rwp)
  - 精修后晶格常数
  - 各相质量分数
- 操作按钮: "开始精修", "保存结果", "查看日志"

**调用函数**:
```python
WPEM.XRDfit(
    wavelength,
    Var,
    Lattice_constants,
    no_bac_intensity_file,
    original_file,
    bacground_file,
    MODEL='REFINEMENT',
    iter_max=40,
    cpu=4
)
```

#### 3.2.5 非晶分析模块

**功能**: 非晶相峰拟合和RDF计算

**子页面1: 非晶峰拟合**
- 输入: 非晶相数据文件
- 参数:
  - 非晶峰数量
  - 2θ范围
  - 初始峰位置 (可固定)
  - 最大迭代次数
- 输出: 拟合曲线, 各峰参数

**子页面2: RDF计算**
- 输入: 非晶相去背景数据
- 参数:
  - 波长
  - 最大半径r
  - 样品密度
  - 突出峰数
- 输出: RDF曲线

**调用函数**:
```python
# 非晶峰拟合
WPEM.Amorphous_fit(
    mix_component,
    amor_file,
    ang_range,
    peak_location,
    max_iter
)

# RDF计算
WPEM.AmorphousRDFun(
    wavelength,
    amor_file,
    r_max,
    density_zero,
    highlight
)
```

**调用函数**:
```python
WPEM.EXAFSfit(
    XAFSdata,
    power=2,
    distance=5,
    k_point=8,
    k=3,
    transform='fourier',
    de_bac=False

```

## 5. 数据流设计

### 5.1 模块间数据传递

```
用户输入 → 参数验证 → 转换为PyXplore格式 → 调用底层函数 → 结果处理 → 界面展示
                     ↓
               错误处理和异常捕获
                     ↓
               写入日志系统
```

### 5.2 文件处理流程

1. **输入阶段**: 用户选择文件 → 验证格式 → 读取数据 → 显示预览
2. **处理阶段**: 设置参数 → 调用PyXplore → 显示进度 → 处理结果
3. **输出阶段**: 可视化结果 → 保存文件 → 生成报告
